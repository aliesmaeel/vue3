## Vue3 projects with composition API 
---
The projects are :

* Add people to list 
* Calendar
* Markdown
* Slider
* Calculator
* Reusable Modal
* Crud 
* Real time chat ( Firebase )
* Objects Detection (Tensor.js)
---
# Technologies used 
1.   Tensorflow
2.   Netlify
3.   Vue3
4.   Vue Router v4
5.   Vuex v4
6.   Firebase Authentication
7.   Firebase Real Time Database

# download 
```bash
npm install
```
After installation, you need to run the command
```bash
npm run dev
```
